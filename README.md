# Sistema de Gestión de Turnos

Autores: Grupo 4 – Ingeniería de Software II  
Docente: Antonio Rodríguez Linares  
Fecha: Noviembre 2025

## Descripción
Aplicación web para reservar y gestionar turnos en consultorios o barberías; permite crear, modificar y cancelar citas. [attached_file:12]

## Instalación
1. Clonar el repositorio  
   git clone https://github.com/usuario/proyecto_turnos.git [attached_file:12]
2. Crear entorno e instalar dependencias  
   pip install -r requirements.txt [attached_file:12]
3. Ejecutar la aplicación (modo local)  
   python app.py [attached_file:12]
4. Ejecutar pruebas  
   pytest -v [attached_file:12]

## Tecnologías usadas
- Python 3.10  
- Flask
